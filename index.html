Генерации и визуализация фрактальных изображений с использованием систем Линденмайера

Программа на Python, которая позволяет отрисовывать фракталы (детерминированные и стохастические). В меню можно выбрать как уже предварительно разработанные системы, так и описать свои. Рассмотрены простейшие фракталы, более сложные, с внесением изменений в изначальную систему, деревья или ветвящиеся фракталы, а также привен стохастический ветвящийся фрактал, имитирующий природную форму дерева.

Реализация методов Гаусса

Проект посвящен реализации методов Гаусса для решения систем линейных уравнений, включая метод последовательного исключения по столбцам, оптимальный метод Гаусса для исключения неизвестных и метод Гаусса-Жордана. Представлены две версии каждого алгоритма: на стандартных списках Python и на списках с индексацией от 1 для удобства работы с матрицами.

Калькулятор, позволюящий выполнять вычисления для набора выражений из файла

Калькулятор поддерживает четыре арифметических действия, позволяет вычислять выражения с заданной точностью. Репозиторий содержит модуль, в котором написан контекстный менеджер для проведения вычислений для набора примеров из текстового файла. Все функции осуществляющие работу программы закомментированы и покрыты тестами с использованием фреймворков pytest и unittest. Для создания программы использовался Python и его библиотеки.

GitHub: ссылка на проект

Численные методы

Репозиторий содержит инструменты для расчета интегралов и решения дифференциальных уравнений. Включает методы прямоугольников, трапеций и Симпсона с возможностью выбора шага, а также алгоритмы для вычисления с заданной точностью. Предоставляет методы Эйлера и Рунге-Кутта для решения дифференциальных уравнений разных порядков и систем, а также приближенные методы для систем нелинейных уравнений и функции для вычисления элементарных функций.

Интерполяция

Репозиторий включает инструменты для вычисления промежуточных значений функций, используя полиномы Лагранжа, Ньютона и Стерлинга, с возможностями визуализации данных.
